NZ State Highway Traffic Analytics

ETL Pipeline  Container Orchestration  IaC  Data Engineering Zoomcamp 

This end-to-end pipeline provides daily traffic count across New Zealand State Highways which is designed to deliver key metrics such as vehicle counts, the most frequented monitoring sites, percent heavy load vehicles. The pipeline started from ingesting data from API, stored it in Google Cloud Storage, then transformed using dbt (data build tool) and uploaded it to data warehouse (Bigquery) for visualisation.
